神明幼稚園 採用特設スマホページ（デモ）

■ 開き方
1. ZIPファイルを右クリックして「すべて展開」を選びます。
2. 展開された「shinmei-recruit-demo」フォルダを開きます。
3. 中にある「index.html」をダブルクリックします。
4. ブラウザ（Edge または Chrome）が開き、デモをご覧いただけます。

■ 操作方法
・右上の MENU を押すと、ページ内の各項目へ移動できます。
・「募集要項を見る」を押すと、デモ用の案内が表示されます。
・「園見学を申し込む」を押すと、申込フォームが開きます。
・フォームはデモです。送信しても実際にはメール送信やデータ保存は行われません。

■ ご注意
・インターネット接続、ローカルサーバー、ターミナルは不要です。
・index.html、style.css、script.js、assets フォルダは同じフォルダ構成のまま保管してください。
・募集条件の内容は確認中／デモ用仮表示です。
